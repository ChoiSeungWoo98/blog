## 리눅스 명령어 정리
ls : 현재 디렉토리 파일 목록 출력(-l : 자세하 보기)
pwd : 현재 경로
mkdir : 폴더 생성
touch : 파일 생성
cd : 디렉토리 변경
rm : 파일 지우기(-r : 안에 있는 모든 폴더, 파일 제거)
rmdir : 폴더 지우기
